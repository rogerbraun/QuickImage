QuickImage

What is this?

A way to get random placeholder images from Google Image Search.

It's heavily inspired by jpg.to.

The thumbnails below are created this way. Try reloading the page.

How to use?

<img src="http://quickimage.heroku.com/image/keyword/width/height" />
This will return a jpg.

So use this: 

<img src="http://quickimage.heroku.com/image/cat/300/200" />
if you want a 300x200 picture of a cat.
